Búsquedas usando find

2016-09-01

Navegar entre archivos es complicado, buscar un archivo en especifico aún más. Para ayudarnos con esta tediosa tarea existe el comando find. Dicho comando nos sirve para buscar archivos en el sistema de forma muy rápida y dando un resultados que podemos personalizar e incluso efectuarle tareas muy especificas. Veamos algunos ejemplos.

Buscar archivos con el nombre “gulpfile” en el directorio actual.

Hay que tener especial cuidado con el comando find ya que puede consumir todo el poder de procesamiento de nuestra máquina. Esto nos dejaría en la penosa necesidad de reiniciar… qué? olvidaron guardar algo? I´m so sorry my friend.

Para evitar quedarnos sin recursos en el computador podemos pasarle ciertos parámetros a nuestra búsqueda para hacerla más especifica.

Especificar nivel de subdirectorios a los cuales buscar.

Se pude buscar por tipo.

  • f: archivo
  • d: directorio
  • l: link simbolico

Buscar por extensión de archivo.

También podemos buscar por tamaño.

  • c: bytes
  • k: Kilobytes
  • M: Megabytes
  • G: Gigabytes

Y claro podemos combinar todo esto.

Nice, right? Así de especificos podemos ser en nuestras busquedas y aún más. Les invito a ver el manual de find man find.
