Klipse Javascript Live Coding

2016-08-24

Anteriormente he utilizado codepen.io para ejemplificar código Javascript pero tiene la desventaja de ser muy pesado y de no permitir el “live coding” dentro de la misma página donde está incrustado.

Es por eso que buscando alternativas para ejemplificar sólo código Javascript me tope con Klipse un “evaluador de código” que es muy fácil de implementar, ligero y permite el live coding.

El siguiente código tiene un error, solucionalo agregando la variable yourName. Algo así:



  function sayHi(name){
    return 'what do you think ' + name + '? nice, right?';  
  }

  sayHi(yourName);

Veamos como implementarlo en tu sitio…

Nota: Klipse soporta otros lenguajes como ruby y php.

Lo puedes implementar en cualquier página agregando lo siguiente al final del body.

Para el editor utiliza CodeMirror el cual se puede personalizar bastante, yo sólo le agregue números de linea y auto cierre de corchetes, paréntesis etc.

Después sólo hace falta que le asignes la clase .language-klipse-eval-js al elemento html que contiene el código.

Yo personalmente he utilizado los elementos <pre><code>CODIGO JAVASCRIPT</code><prev>, <pre> porque este blog usa markdown para generar el html y tenía problemas con las comillas y <code> simplemente por semántica, bien podría ser un <div>.


  //multiply function
  function multiply(a, b) {
    return a * b;
  }

  multiply(20, 234)

Y así tenemos un editor de Javascript que evaluá el código en nuestro sitio.
